###    Simple Text Processor

A simple Java console application that performs basic text-processing operations:

Counts the number of words in a sentence

Prints each word on a new line

Reverses a given string

The project is built using Maven and follows a minimal setup suitable for demo purposes.

### How to Run
1. Clone the repo:
```bash
git clone https://github.com/lukesh-b/java-prjs.git
cd java-prjs

```

2. Build using Maven:

```bash
mvn clean compile
```
=> This will compile the source code and place the compiled .class files in the target directory.

3. Run the Application on compiled

```bash
java -cp target/classes text_processing.TextProcessingUtils
```
Note: Fully qualified class name must be used when running.
